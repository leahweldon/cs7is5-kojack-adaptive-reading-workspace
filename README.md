# CS7IS5 – Adaptive Applications (Team Kojack)

## Project
**Scrutable Adaptive Reading Workspace**  
A reading workspace that adapts text presentation and support features (e.g., chunking, bionic reading, glossary prompts) based on user preferences and interaction, while making adaptations transparent and user-controllable.

## Key Features (POC)
- User preferences (support level + toggles)
- Bionic reading mode
- Chunking / focus view
- Glossary / term support (basic)
- Scrutability side panel: **Your Model**, **Why this changed**, **Controls**
- Session summary and lightweight feedback

## Tech Stack (planned)
- React (frontend)
- Storage: localStorage or SQLite (TBD)
- PDF: pdf.js (TBD)
