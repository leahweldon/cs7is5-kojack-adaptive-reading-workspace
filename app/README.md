# Adaptive Reading App

## Quick Start

1. Install Node.js 20+.
2. From the `app` folder, run:

```bash
npm run dev
```

The first run will automatically install all dependencies, then start Vite.

## Useful Commands

```bash
npm run dev      # Auto-installs deps if needed, then starts local dev server
npm run build    # Auto-installs deps if needed, then builds for production
npm run lint     # Auto-installs deps if needed, then runs ESLint
npm run preview  # Preview production build
```

## Notes

- Keep `package-lock.json` committed so every machine installs consistent versions.
- If environment variables are required for auth, see `CLERK_SETUP.md`.
